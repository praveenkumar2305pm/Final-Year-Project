# Final Year Project

## Project Title
Smart Plant Health Monitoring and Management System

## Description
This project is an AI-based system that helps in identifying plant diseases
and recommending suitable fertilizers and remedies.
It uses Machine Learning and Deep Learning techniques to improve crop health.

## Features
- Plant disease detection using CNN
- Fertilizer recommendation system
- Image-based analysis
- Accurate and fast predictions

## Technologies Used
- Python
- Machine Learning
- Deep Learning (CNN)
- Jupyter Notebook
- Git & GitHub

## How to Run
1. Clone the repository
2. Open the Jupyter Notebook files
3. Run the cells step by step
